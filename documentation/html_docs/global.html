attackLocal(row, col, attackArr, button)

Function is called when a player selects a square to attack. Depending on who's turn it is, the opposing player's ship configuration is checked for the existence of a ship piece, marked with a -1 on a hit, and the grid is modified to reflect the hit, by the coloring of a button. If a ship is at the chosen location and all ship pieces have been hit, the player is notified of their victory. Otherwise, the opposing player's screen is switched to after a 3 second timeout.
Parameters:
Name Type Description
row number row of the opposing player's ship array
col number col of the opposing player's ship array
attackArr Object double array representing the attacking player's grid of attacks
button Object the button to modify after a miss or a hit

canPlace(row, col, arr, numPieces, numShips) → {boolean}

Function is called when user attempts to click on a button on the ship selection grid. If the user is allowed to select this position, true is returned, otherwise false. Function checks number of ship pieces remaining to be placed in order to determine the orientation of the ship and whether a ship can actually be placed at this location.
